
## ✅ Проверка TaskTest (mock tasks stream)

После запуска dev-сервера откройте страницу `http://localhost:<vitePort>/task-test`.

Там доступны две вкладки:
- **Tasks**: таблица задач с фильтром по статусу; задачи непрерывно создаются/изменяются/удаляются мок‑агрегатором.
- **Stats**: счётчики и распределение задач по статусам, обновляются в реальном времени.
